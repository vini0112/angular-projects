
## 🚀 Getting Started 
- ℹ️ Make sure to have cloned the repository in the first README! If you don't  -> 🔗 [MAIN README](../README.md)
  
### 1. get into the frontend folder
   ```
  cd api_clothe-store
  ```
### 2. Install the dependecies
   ```
  npm install
  ```
### 3. Run the Project
   ```
  ng start
  ```
